- Slouží jako **opozice hráčů** (nebo jako neutrální síla)
- Frakce = skupina, která má společný cíl
- **Základní otázky** při vytváření frakce
	- **Identita** - role ve světě
	- **Místo vlivu** - Kde, kdy, kdo
	- **Síla** - jak velkou silou vládne
	- **Ideologie** - principy a hodnoty
# Běžné organizace
## Vlády
- Cíle: stabilita, řád, sebezáchova, status quo

| **Cíle hráčů**                                                                                                                                       | **Cíle vládní frakce**                                         |
| ---------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| Ulovit mocné monstrum, profitovat z obchodu, založit osadu.                                                                                          | Zvýšit bezpečnost místních cest a obchodních tras.             |
| Zahájit povstání rolníků, svrhnout zkorumpovaného vládce, účastnit se odborové organizace.                                                           | Potlačit protistátní nálady mezi nižšími vrstvami.             |
| Bojovat v určité válce, vyhnout se odvodu, obsadit území.                                                                                            | Shromáždit vojáky pro místní armádu nebo milici.               |
| Poskytnout pomoc při válce nebo katastrofě, způsobit válku nebo katastrofu, profitovat z války nebo katastrofy.                                      | Zorganizovat místní frakce pro válečné nebo humanitární úsilí. |
| Vyhrát spor proti určité osobě, profitovat z právního sporu, poskytnout bezpečnostní nebo vojenskou sílu někomu v právním sporu.                     | Urovnat právní nebo sociální spor mezi dvěma stranami.         |
| Poskytnout pomoc potřebným, profitovat z tržní manipulace, profitovat z prodeje předmětů nalezených při dobrodružství.                               | Řešit ekonomickou krizi.                                       |
| Získat přístup do omezených oblastí, shromáždit informace o důležité osobě, získat přístup k určité vládní frakci.                                   | Navázat příznivý diplomatický vztah s jinou vládní frakcí.     |
| Činnost zakázaná novým zákonem byla povolena starým zákonem. Profitovat z lovu lidí, kteří porušují nový zákon.                                      | Vynutit nový zákon.                                            |
| Dosadit určitou osobu jako vládce, bránit současného vládce nebo vládní orgán, získat nezávislost na vládě.                                          | Konsolidovat moc pod současným vládcem nebo vládním orgánem.   |
| Profitovat z obchodu určité oblasti nebo zboží, bránit určitý náklad nebo obchodní trasu, získat přístup k určitému obchodnímu nebo vládnímu frakci. | Zajistit příznivou obchodní dohodu s jinou vládní frakcí.      |
## Gildy, odbory, kartely..
- Zastupují zájmy pracovníků a řemeslníků
- Obecně slabší a méně organizované než vlády
- Postavy jsou často členy
- Cíle: Hájí zájmy svých členů

| **Cíle hráčů**                                                                                                                | **Cíle pracovní frakce**                                                   |
| ----------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| Získat velký architektonický projekt, pomoci přesunout zásoby, pomoci nebo poškodit kriminální organizaci.                    | Zajistit pracovní smlouvu od mocnější frakce (vládní/kriminální).          |
| Bránit pracovní místo, prozkoumat nebezpečné místo, potlačit populární hnutí.                                                 | Zajistit lepší pracovní podmínky (méně nebezpečné/více pracovních ochran). |
| Získat přístup do omezeného společenského okruhu, splatit laskavost někomu bohatému, využít vztah pro zisk.                   | Vyjednat obchodní dohodu, která pomůže členům dosáhnout zisku.             |
| Získat určitý předmět nebo vzácný zdroj, prodat neobvyklý předmět, najít vzácné suroviny nebo přísady pro rituál nebo kouzlo. | Získat monopol na určitý předmět nebo vzácný zdroj.                        |
| Ožebračit bohatého člověka nebo organizaci, získat majetek produkující bohatství, snížit korupci.                             | Manipulovat s trhem určité komodity.                                       |
| Pomstít se, dosadit spojence na pozici moci, převzít frakci.                                                                  | Svrhnout nebo nahradit vlastní vedení.                                     |
| Vytvořit koalici frakcí, eliminovat konkurenci, zmobilizovat velké množství lidí pro nějakou věc.                             | Spojit se s, absorbovat nebo zničit související frakci.                    |
| Získat přístup do omezené oblasti, zajistit pomoc vůdce frakce, podkopat vládní frakci.                                       | Zrušit nebo přijmout zákon, který prospěje frakci.                         |
| Šířit náboženské hodnoty, získávat informace od místních, profitovat z krizové situace.                                       | Přímo pomoci svým vlastním členům v krizové situaci.                       |
| Infiltrujte nepřátelskou organizaci, získejte přístup do omezené oblasti, najděte kupce pro nezákonné zboží.                  | Navázat obchodní partnerství se známým nepřítelem jiných mocných frakcí.   |
## Zločin
- Obvykle v opozici Vládě
- Obvykle se skrývá
- Cíle: peníze a moc
	- důležitější je, jaké metody používají

| **Cíle hráčů**                                                   | **Cíle kriminální frakce**                                                      |
| ---------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| Získat vzácnou přísadu nebo odborníka z určité části města.      | Zahájit nové ochranné výpalné v této části města.                               |
| Získat vzácný řemeslný materiál pro osobní projekt nebo patrona. | Přepadnout karavanu přepravující vzácné řemeslné materiály.                     |
| Pomoci politikovi přelstít rivaly.                               | Vydírat politika za peníze a laskavosti.                                        |
| Najít nebezpečného obchodníka se zbraněmi.                       | Zajistit výhodnou dohodu s černým trhem se zbraněmi.                            |
| Zničit nebo oslabit kriminální frakci.                           | Pomstít se kriminální frakci za urážku nebo násilný incident.                   |
| Zajistit (exotické) zvíře                                        | Propašovat nebezpečné zvíře do města.                                           |
| Odhalit pachatele zločinu a přivést ho před spravedlnost.        | Zaplatit úplatek orgánům činným v trestním řízení, aby ignorovaly tento zločin. |
| Půjčit si cenný předmět z dobře chráněného trezoru.              | Vykrást slavně chráněný trezor.                                                 |
| Získat mocný magický artefakt.                                   | Prodat padělek jako pravý.                                                      |
| Získat starožitný předmět od sběratele na aukci.                 | Ukrást zboží z aukce, zatímco všechny cennosti jsou pod jednou střechou.        |
## Náboženství
- Následovníci božstev, kulty, církve
- Mohou hrát roli spojenců i opozice
- Cíle: Různorodé v závislosti na bohovi/božstvu

| **Slouží bohu…**      | **Cíle náboženské frakce**                                                                                              |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| Zdraví, charita.      | Poskytovat pomoc, přístřeší, jídlo a léky chudým a nemocným.                                                            |
| Síla, válka.          | Podporovat místní armádu, obracet vojáky na víru, posvěcovat zbraně svatou energií.                                     |
| Láska, loajalita.     | Oddávat manželství, podporovat loajalitu, trestat zrádce.                                                               |
| Vědění, intelekt.     | Získávat vzácné knihy pro katedrální knihovnu, financovat místní školy.                                                 |
| Pomsta, spravedlnost. | Trestat zlé, veřejně odhalovat zločince, vykořenit zdroje korupce.                                                      |
| Zvrhlost, nenávist.   | Zajišťovat oběti pro bohy, vyvolávat démony, masakrovat soupeřící členy náboženství.                                    |
| Krása, dokonalost.    | Podporovat místní malíře a umělce, prosycovat oblast svatým kouzlem, vytvářet dokonalé artefakty se vzácnými přísadami. |
| Cestování, obchod.    | Posílat kněze bezpečně na poutě, podporovat spojenecké obchodní gildy, získávat přesné mapy.                            |

| **Cíle hráčů** | **Cíle náboženské frakce** |
| --- | --- |
| Získat magické léčení. | Získat zpět léčivý pramen od bandy odporných démonů. |
| Pomoci vést válku proti nepřátelské frakci. | Očarovat zbraně křižáků, kteří se chystají do války, svatým olejem, který je v nedostatku. |
| Získat důvěru místního šlechtice nebo oblíbené veřejné osoby. | Zrušit kletbu na místním šlechtici nebo oblíbené veřejné osobě. |
| Zničit reputaci náboženské frakce. | Utajit znalosti o minulých herezích jejich frakce. |
| Najít kupce pro určený artefakt z ruin chrámu. | Najít relikvii svatého. |
| Zničit reputaci místního šlechtice. | Veřejně dokázat, že místní šlechtic je kacíř. |
| Najít ztraceného člena rodiny. | Získat oběti pro nadcházející krvavý rituál. |
| Najít ztraceného umělce. | Objednat umělecké dílo nebo ikonografii od známého umělce. |
| Vykonat pouť na svaté místo v nebezpečném území. | Zřídit odpočívárnu na podporu poutníků v nebezpečném území. |
| Přesvědčit místní vládní osobnost, aby přijala způsob myšlení určité frakce. | Obrátit místní vládní osobnost na jejich náboženské přesvědčení. |
# Jak pracovat s frakcemi ve hře
- cca 4 frakce jsou aktivní v jeden okamžik
- cca 6 frakcí operují na jedné lokaci
- Každá nová frakce by měla mít:
	- **Jméno**
	- **Relativní moc**
	- **Lokaci**
	- **Krátký popis**
	- **1-2 dlouhodobé cíle** (použij "hodiny")
